```
https://nodejs.org
```

```
https://git-scm.com
```

### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone https://povdai.github.io/55-grupe-bandomasis/
    ```
2. Install main NPM packages
    ```sh
    npm i
    ```
3. Install sub-directories NPM packages
    ```sh
    npm run install-all
    ```
4. Run the servers
    ```sh
    npm run dev
    ```
